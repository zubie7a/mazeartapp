## MaZeArtApp
Random pattern generator inspired by [Hitomezashi Stitch Patterns](https://www.youtube.com/watch?v=JbfhzlMk2eY), with multiple parameters for resolution, symmetries, and palettes.

Try it at [https://zubie7a.github.io/mazeartapp/](https://zubie7a.github.io/mazeartapp/)

Read more [https://www.z10z.xyz/post/random-pattern-generator/](https://www.z10z.xyz/post/random-pattern-generator/)
